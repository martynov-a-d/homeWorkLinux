Команда cd используется для перемещения внутри файловой системы
Синтаксис: 
	cd адрес/файл (относительный путь)
	cd ~/адрес/файл (полный путь)
	cd (позволяет быстро перейти в домашний каталог)
	cd / (позволяет быстро перейти в корневой каталог)
	cd .. (подняться на уровень выше)
	cd - (позволяет венуться в предыдущую директорию(очень, очень полезная программа))

Команда mkdir используется для создания папки или директории папок
Синтаксис:
	mkdir имя_папки (создает папку с указанным именем)
	mkdir имя_папки имя_папки имя_папки (создает несколько папок с указанными именами)
	mkdir -p /имя_папки/имя_папки (создает директорию с указанным именем и ее родителей)

Команда touch используется для взаимодействия с файлом
Синтаксис:
	touch имя_файла (создаст файл с указанным именем)
	остальные напишу когда полностью пойму зачем они мне :)

Команда cp используется для создания копии и работы с копируемым объектом
Синтаксис:
	cp имя_файла куда_копировать (создает копию данного файла)
	cp -R имя_папки куда_копировать (создает копию данной папки)
	cp имя_файла куда_копировать/имя_файла (создает копию данного файла и присваивает указанное имя)

Команда mv используется для перемещения объектов
Синтаксис:
	mv имя_файла куда_перенести (переместить данный файл)
	mv имя_папки куда_переместить (переместить данную папку)
	mv имя_файла имя_файла (переименовать данный файл)
	mv имя_папки имя_папки (переименовать данную папку)

Команда ssh позволяет подключаться к серверу 
Синтаксис:
	ssh user_name@host (соединение с сервером под указаным лог)
	
Настройка:
	ssh-keygen генерируем ключ
	ssh-cope-id user_name@host
